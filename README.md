# Biblioteca API

Este projeto simula uma API para gerenciamento de biblioteca.

O objetivo é demonstrar a organização de um projeto Back-end
utilizando GitHub.

## Estrutura do Projeto

src/
arquivos que simulam a lógica da API

docs/
documentação do projeto e rotas

config/
configurações da aplicação

tests/
testes simulados da API

## Rotas simuladas

GET /livros
POST /livros
PUT /livros/:id
DELETE /livros/:id
